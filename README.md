# estp.jp

ESTP向けの診断・ゲーム・キャラクターコンテンツをまとめたサイトです。

## About

estp.jp is a Japanese web project focused on MBTI-style personality content, especially for ESTP users.

## Features

- ESTP向け診断コンテンツ
- MBTI系のゲーム・企画
- キャラクター・説明書コンテンツ
- スマホ対応のシンプルなWebサイト

## Tech Stack

- HTML
- CSS
- JavaScript
- Firebase / Vercel など必要に応じて

## Status

個人制作・実験中のプロジェクトです。
